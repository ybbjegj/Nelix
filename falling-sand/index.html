📖 游戏说明

🖱️ 基本操作

  • 鼠标左键 / 触屏拖动 — 绘制当前选中的材料
  • 滚轮 或 [] — 调整画笔大小
  • 空格 — 暂停 / 继续模拟
  • 1~0,-,= — 快速切换材料
  • H — 打开/关闭帮助面板

🧱 材料一览

  • 🟡 沙子 — 受重力下落的颗粒，可堆积成沙堆
  • 🔵 水 — 流体，向下和两侧流动
  • ⬜ 石头 — 不受重力影响的固体，可当墙壁使用
  • 🔥 火焰 — 向上飘动，点燃可燃物，会随时间熄灭
  • 🟫 木头 — 不动的固体，可被火焰点燃
  • ⬛ 油 — 比水轻的液体，极易燃烧
  • ☁️ 蒸汽 — 向上飘散的气体，一段时间后凝结为水
  • 💨 烟雾 — 向上飘散后消失
  • 🌋 熔岩 — 高温液体，点燃可燃物，遇水变蒸汽+石头
  • 🌿 植物 — 接触水会生长，可被火焰点燃
  • 🧪 酸液 — 溶解接触到的大部分材料
  • 🧹 橡皮擦 — 擦除画面上的粒子

⚗️ 材料交互

火 + 木头 → 木头燃烧成火+烟雾
火 + 油 → 油被点燃
火 + 水 → 火熄灭变蒸汽
火 + 植物 → 植物燃烧
熔岩 + 水 → 蒸汽 + 石头
熔岩 + 可燃物 → 可燃物着火
水 + 植物 → 植物生长蔓延
酸液 + 任意材料 → 溶解该材料
蒸汽（冷却后） → 凝结回水
沙子 + 液体 → 沙子沉入液体底部

🏆 挑战模式

点击「挑战模式」按钮进入关卡选择界面，完成各种有趣的物理挑战！每一关都有不同的目标和玩法。完成后会显示🌟标记。

💡 小提示

  • 用石头画容器，再往里灌水或沙子，观察物理效果
  • 试试在油面上点火，观看壮观的连锁燃烧
  • 翻转重力可以创造有趣的反向瀑布效果
  • 植物在水旁边会自动生长蔓延
